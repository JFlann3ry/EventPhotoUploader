Frequently Asked Questions

How do I upload my photos?

Click the "Guest Login" link or scan the event QR code. Enter your email, select your photos ro videos, and click "Upload".

Do I need to create an account?

No account is required for guest uploads. You can upload photos directly using the event code and password provided by the event organiser.

What file types can I upload?

You can upload common image formats such as JPG, PNG, and GIF. Some events may also allow video uploads.

Is there a limit to the number of photos I can upload?

There may be a limit set by the event organiser. If you encounter an upload limit, please contact the event host.

Who can see my uploaded photos?

Only the event organiser can access the event gallery to view guests uploads. Your photos are private and secure.

How do I report inappropriate content?

Please use the Contact Us page to report any issues or inappropriate content.

I forgot the event code or password. What should I do?

Contact your event organiser to retrieve your event code or password.
